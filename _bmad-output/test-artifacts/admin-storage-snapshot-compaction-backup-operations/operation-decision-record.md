## Implementation Direction

This story will not add fake EventStore upstream engines. Admin.Server command services will return typed honest-defer results without DAPR service invocation for the listed operations, and controller/API/UI code will preserve those results so users see a clear deferred outcome instead of "Admin service unavailable" or a 404 caused by absent upstream routes.
