## Key Takeaways

- Testing is an engineering discipline, not an afterthought
- P0-P3 matrix prioritizes test coverage based on risk (Probability x Impact)
- TEA Definition of Done has 7 quality standards that prevent test maintenance nightmares
- Hard waits (`sleep`) violate DoD because they don't react to state — use `waitFor` instead
- As a Tech Lead, enforce these standards in code review — flaky test PRs don't merge
