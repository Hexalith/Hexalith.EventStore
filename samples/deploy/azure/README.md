# Azure Container Apps — Supplementary Deployment Artifacts

Reference Bicep modules that supplement the Aspire ACA publisher output. The Aspire publisher (`PUBLISH_TARGET=aca`) is the **primary** deployment path.

These files provide DAPR component definitions and managed identity role assignments not included in the Aspire-generated Bicep.

**Full walkthrough:** [Azure Container Apps Deployment Guide](../../../docs/guides/deployment-azure-container-apps.md)
