# API Contracts — Hexalith.EventStore

> All HTTP endpoints derived from the code. Two HTTP surfaces: the **gateway** (`src/Hexalith.EventStore`)
> and the **admin API** (`src/Hexalith.EventStore.Admin.Server`). See the hand-authored
> `docs/reference/command-api.md` and `docs/reference/query-api.md` for narrative reference, and
> `docs/reference/problems/` for the RFC 9457 problem catalog.

## Conventions

- Base path `/api/v1/`. Content-Type `application/json`; errors `application/problem+json` (RFC 9457).
- JWT bearer required unless marked `[AllowAnonymous]`. UserId from `sub` claim; tenant from `eventstore:tenant` claim(s).
- Global request body limit **1 MB** (Kestrel). Per-tenant + per-consumer rate limiting (429 + `Retry-After`).
- `X-Correlation-ID` honored/generated by `CorrelationIdMiddleware` (alphanumeric+hyphen, ≤128 chars).
- Identifiers are **ULIDs**; ID fields validated by regex (alphanumeric + hyphen), not `Guid.TryParse`.

## Gateway API (`src/Hexalith.EventStore`)

### Commands

| Method | Route | Controller | Auth | Notes |
|--------|-------|------------|------|-------|
| `POST` | `/api/v1/commands` | `CommandsController.Submit` | JWT | Submit command. Body `SubmitCommandRequest`. **202** + `Location: api/v1/commands/status/{correlationId}`. Result payload capped 64 KB. |
| `GET` | `/api/v1/commands/status/{correlationId}` | `CommandStatusController.GetStatus` | JWT | Poll status. In-flight states (`Received/Processing/EventsStored/EventsPublished`) return `Retry-After`; terminal states (`Completed/Rejected/PublishFailed/TimedOut`) do not. Searches all authorized tenants. |
| `POST` | `/api/v1/commands/replay/{correlationId}` | `ReplayController.Replay` | JWT | Replay an archived failed command. Only `Rejected/PublishFailed/TimedOut` replayable (else **409**). **202** + new ULID correlation id. |
| `POST` | `/api/v1/commands/validate` | `CommandValidationController.Validate` | JWT | Preflight (dry-run) tenant+RBAC auth. **200** + `PreflightValidationResult` (`IsAuthorized=false` on denial, not 403). |

**`POST /api/v1/commands` status codes:** 202, 400, 401, 403, 404, 409, 422, 429, 503.

### Queries

| Method | Route | Controller | Auth | Notes |
|--------|-------|------------|------|-------|
| `POST` | `/api/v1/queries` | `QueriesController.Submit` | JWT | Execute query. Honors `If-None-Match` → **304** (Gate 1). Body `SubmitQueryRequest` (paging/search/filters/orderby/freshness skip cache). Sets ETag response header. |
| `POST` | `/api/v1/queries/validate` | `QueryValidationController.Validate` | JWT | Preflight query auth → **200** `PreflightValidationResult`. |

**`POST /api/v1/queries` status codes:** 200, 304, 400, 401, 403, 404, 429, 503.

### Streams

| Method | Route | Controller | Auth | Notes |
|--------|-------|------------|------|-------|
| `POST` | `/api/v1/streams/read` | `StreamsController.ReadStreamAsync` | JWT | Read an event-stream page. Body `StreamReadRequest` (`FromSequence`, optional `ToSequence`, `PageSize` 1–1000). Returns `StreamReadPage` (events + metadata, `Truncated` flag). Offset pagination only (continuation tokens rejected). Returns protected-data readability decision on unreadable events. |

**Status codes:** 200, 400, 401, 403, 404, 500, 503. See also `docs/reference/stream-replay-api.md`.

### Projection notifications (internal)

| Method | Route | Controller | Auth | Notes |
|--------|-------|------------|------|-------|
| `POST` | `/projections/changed` | `ProjectionNotificationController.OnProjectionChanged` | `[AllowAnonymous]` | DAPR pub/sub subscription `*.*.projection-changed`. Regenerates ETag via `IETagActor`, broadcasts to SignalR (fail-open). Non-200 triggers DAPR retry. |

> **Note on `[AllowAnonymous]` admin routes:** these are reachable via DAPR access-control + the
> composite "Hexalith" scheme (trusted `dapr-caller-app-id`). In self-hosted/dev mode DAPR access
> control is allow-by-default; production requires deny-by-default + mTLS (see deployment-guide.md).

### SignalR

- Hub `ProjectionChangedHub` mapped at `ProjectionChangedHub.HubPath` (config-gated). Server broadcasts
  `ProjectionChanged(projectionType, tenantId)` to group `{projectionType}:{tenantId}`; clients call
  `JoinGroup`/`LeaveGroup`. Redis backplane via `SignalR.StackExchangeRedis`. SignalR runtime-proof
  endpoints are mapped when enabled. See `docs/operations/redis-signalr-channel-isolation.md`.

### OpenAPI / health

- `MapOpenApi()` at `/openapi/v1.json` + Swagger UI at `/swagger` (gated by `EventStore:OpenApi:Enabled`).
- `MapErrorReferences()` → human-readable `/problems/{errorType}` pages.
- `MapApiVersionFallback()`, `MapSubscribeHandler()` (DAPR pub/sub), `MapActorsHandlers()` (DAPR actors).
- Health: `/health`, `/alive`, `/ready` (exempt from rate limiting).

## Admin API (`src/Hexalith.EventStore.Admin.Server`, host = `eventstore-admin`)

REST API consumed by Admin.UI, Admin.Cli, and Admin.Mcp. Backed by DAPR actors/state, delegating writes
to the gateway via DAPR service invocation (ADR-P4). 10 controllers, 3-tier RBAC policies
(`AdminReadOnly` / `AdminOperator` / `AdminFull`):

| Controller | Surface | Min policy |
|------------|---------|-----------|
| `AdminStreamsController` | timeline, state reconstruction, diff, blame, bisect, causation/correlation, sandbox | ReadOnly (read) |
| `AdminProjectionsController` | status, pause/resume/reset/replay | Operator (writes) |
| `AdminStorageController` | storage overview, snapshot policies, manual snapshot, compaction | Operator |
| `AdminDeadLettersController` | query, resubmit, purge | Operator |
| `AdminConsistencyController` | trigger/cancel checks, anomalies | Operator |
| `AdminBackupsController` | backup/restore (point-in-time), stream export/import, validation, admission (22.7c), crypto-shredding | Full |
| `AdminTenantsController` | tenant CRUD, enable/disable, user role management | Full |
| `AdminHealthController` | system + component health | ReadOnly |
| `AdminDaprController` | actors, components, subscriptions, resiliency metadata | ReadOnly |
| `AdminTypeCatalogController` | aggregate/command/event type catalog | ReadOnly |
| `AdminTracesController` | correlation trace mapping | ReadOnly |

> Admin API also serves OpenAPI/Swagger (gated by `EventStore:Admin:OpenApi:Enabled`).
> See `docs/reference/admin-stream-export.md` and `docs/operations/admin-debugging-json-large-stream-hardening.md`.

## RFC 9457 Problem Types (selected)

`validation-error` (400), `authentication-required`/`token-expired` (401), `forbidden` (403),
`not-found`/`command-status-not-found` (404), `concurrency-conflict` (409), domain-rejection sub-URIs (422),
`rate-limit-exceeded`/`backpressure-exceeded` (429), `service-unavailable` (503), `not-implemented` (501),
plus `restored-backup-admission-conflict`, `crypto-shredding-workflow-conflict`, `unreadable-protected-data`.
Extension keys: `correlationId`, `tenantId`, `errors`, `reason`, `reasonCode`, `retryAfter`,
`rejectionType`, `correctiveAction`, `degradation` (`GatewayProblemDetailsExtensions`). Full catalog:
`docs/reference/problems/index.md`.
