## Project Overview

- **Type:** Monorepo / multi-part (event-sourcing **backend** + Blazor admin/sample UI tier)
- **Primary language:** C# (.NET 10, SDK 10.0.300)
- **Architecture:** CQRS + DDD + Event Sourcing on DAPR actors & pub/sub, orchestrated by .NET Aspire
